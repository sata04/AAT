# 概要 - AAT (Acceleration Analysis Tool)

## AATとは

**AAT (Acceleration Analysis Tool)** は、微小重力環境下での実験データを解析・可視化するための専用デスクトップアプリケーションです。宇宙実験や落下塔実験などで取得した加速度データを読み込み、重力レベルの計算、統計分析、グラフ表示を簡単に行うことができます。

AATは、研究者やエンジニアが微小重力実験の品質を評価し、実験結果を効率的に分析するために開発されました。

> [!NOTE]
> 📸 **スクリーンショット予定地**: AATメインウィンドウの全体像（データ読み込み後のグラフ表示状態）を追加予定

## ✨ 主な特徴

### 📊 簡単なデータ分析

- CSVファイルをボタンクリックで読み込み
- 列の自動検出により、手動設定不要（必要に応じて手動選択も可能）
- Inner CapsuleとDrag Shieldの両方のデータを同時に処理

### 📈 リアルタイム可視化

- データ読み込み後、即座にグラフを表示
- Inner CapsuleとDrag Shieldの重力レベルを同時プロット
- 高品質なMatplotlibベースのグラフ

### 🔍 詳細統計分析

- **G-quality評価**: 複数のウィンドウサイズ（0.1秒〜1.0秒）での微小重力環境の品質評価
- **最小標準偏差検出**: 最も安定した時間窓を自動的に特定
- **統計情報の自動計算**: 平均値、標準偏差、範囲など

### 💾 自動データ保存

- Excel形式（`.xlsx`）での総合分析結果の出力
- グラフ画像（PNG形式）の自動保存
- 元のCSVデータも含めてエクスポート

### ⚡ 高速処理

- **インテリジェントキャッシュ**: 一度処理したデータを再利用
- ファイル変更時の自動再処理
- バックグラウンド処理によるUIの応答性維持

### 🖱️ インタラクティブ機能

- **範囲選択**: グラフ上でマウスドラッグして特定範囲を選択
- **リアルタイム統計**: 選択範囲の統計情報を即座に表示
- **比較モード**: 複数のデータセットを同時に表示して比較

### 🎨 モダンなUI

- **テーマ切替**: ダークモード、ライトモード、システムデフォルト
- **レスポンシブデザイン**: 画面サイズに合わせて最適化
- **直感的な操作**: 初めてのユーザーでも迷わず使える設計

## 📋 ユースケース

### 1. 落下塔実験の品質評価

落下塔実験で取得した加速度データを読み込み、微小重力環境の品質（G-quality）を評価します。複数のウィンドウサイズで分析することで、どの時間帯で最も安定した微小重力環境が得られたかを特定できます。

### 2. 宇宙実験データの分析

国際宇宙ステーション（ISS）などで実施された実験の加速度データを分析し、実験条件の評価や論文用のグラフを作成します。

### 3. 複数実験の比較

異なる条件で実施された複数の実験データを比較モードで同時表示し、実験条件による差異を視覚的に確認できます。

### 4. リアルタイムデータモニタリング

実験中または実験直後にデータをクイックチェックし、実験の成功/失敗を即座に判断できます。

### 5. バッチ処理

複数のCSVファイルを連続して処理し、結果を自動的にExcelファイルとグラフ画像として保存します。

## 🔧 技術スタック

- **言語**: Python 3.10+
- **GUIフレームワーク**: PySide6 (Qt)
- **データ処理**: NumPy, Pandas
- **グラフ表示**: Matplotlib
- **データエクスポート**: openpyxl, pytables
- **ビルドツール**: PyInstaller (macOSアプリバンドル作成用)

## 📦 出力ファイル

AATは以下のファイルを自動的に生成します:

### Excelファイル (`.xlsx`)

- **Gravity Level Data**: 時間と重力レベルのデータ
- **Gravity Level Statistics**: 統計情報（平均、標準偏差など）
- **Acceleration Data**: 元の加速度データ
- **G-quality Analysis**: G-quality評価結果（実行時のみ）

### グラフ画像 (`.png`)

- **重力レベルグラフ** (`*_gl.png`): Inner CapsuleとDrag Shieldの重力レベル推移
- **G-quality分析グラフ** (`*_gq.png`): 複数ウィンドウサイズでのG-quality評価結果

### キャッシュファイル

- **Pickleファイル** (`.pickle`): 処理済みデータのキャッシュ
- **HDF5ファイル** (`.h5`): 生の加速度データのキャッシュ
